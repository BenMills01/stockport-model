Method

This board ignores availability and financial value. It ranks role-matched players only, requires at least {{ minimum_minutes }} season minutes, and applies a softer minutes-evidence multiplier so tiny samples do not dominate. On-pitch score is 60% technical (role fit, current performance, projection) + 40% physical (peer-percentile SkillCorner output). Football-facing board scores are then adjusted by league strength so cross-league comparisons stay sensible. The physical layer now uses the selected profile template{% if selected_physical_label %} ({{ selected_physical_label }}){% endif %} rather than one shared athletic model for every role. Players without SkillCorner data fall back to technical only.
